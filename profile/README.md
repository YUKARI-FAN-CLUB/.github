# 关于小缘
小缘，二次元著名唱见，也是 Bilibili 的主播，以日语歌曲翻唱为主，已在 Bilibili、网易云音乐、AcFun、5SING 等平台发布了上百个作品。

唱歌声线多变，本音为甜美的少女音，声线可分裂出可爱的萝莉音和沉稳的御姐音，亦因此而受到关注。擅长高音域的歌曲，不论是感情丰富还是爆发力强，都能轻易驾驭。
